# 🍔 Python Food Ordering System

A simple command-line food ordering program built with Python.  
The app displays a menu, allows users to select items, stores them in a cart, and calculates the total price.

## 🚀 Features
- Displays a formatted food menu with prices
- User selects items by name
- Shopping cart system
- Calculates total bill automatically
- Exit anytime by pressing "q"

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run

1. Make sure Python is installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python menu.py
